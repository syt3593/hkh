# NutriLens 项目总结文档

## 1. 项目简介
**NutriLens (AI 智能膳食分析)** 是一个基于 React 和 Google Gemini AI 的现代化 Web 应用，通过图像识别技术帮助用户精准记录和分析每日饮食的营养摄入。

## 2. 技术架构 (Technical Architecture)
- **前端框架**: React 19 + TypeScript
- **构建工具**: Vite
- **AI 引擎**: Google Generative AI (Gemini 2.0 Flash/Pro)
- **样式方案**: Tailwind CSS
- **本地存储**: localStorage (用于存储历史、常餐和偏差样本)
- **关键 API**:
    - **Camera API**: 实时拍摄食物。
    - **Web Speech API**: 语音录入补充信息。
    - **Canvas API**: 图片前端压缩。

## 3. 核心功能 (Core Features)
- **智能识餐**: 拍照或上传照片，AI 自动识别食物单品、预估重量及营养成分。
- **营养看板**: 可视化展示热量、蛋白质、碳水、脂肪等核心数据。
- **交互调节**: 支持滑动调节进食比例，一键 +10%/-10% 微调重量。
- **常餐模版**: 保存高频餐食，实现秒级记录。
- **偏差库**: 自动捕捉 AI 估算大偏差样本，用于模型迭代参考。
- **健康贴士**: 随机展示科学饮食冷知识。

## 4. 当前现状
项目目前为纯前端实现，所有数据均存储在用户浏览器本地。为了支持多端同步、更安全地管理 API Key 以及处理大规模图片存储，建议迁移至后端架构。
